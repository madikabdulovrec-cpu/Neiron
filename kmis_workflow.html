Вход в КМИС (Damumed)
1 Открыть платформу Damumed

Система доступна через веб-браузер. Каждый сотрудник имеет индивидуальный логин и пароль.

↓
2 Авторизация

Ввести персональные учетные данные. После входа открываются доступные модули в зависимости от роли сотрудника.

↓
3 Выбор модуля «Регистратор»

Основной модуль для записи пациентов. Отображается список врачей с расписанием и доступными слотами.

2
Типы приема пациентов

ОСМС Бесплатный прием

Прием по обязательному социальному медицинскому страхованию. Пациенты прикреплены к поликлинике.

  • Доступен только у врачей с правом приема по ОСМС
  • Пациент должен быть прикреплен
  • Запись возможна самостоятельно через приложение Damumed

Платный Платные услуги

Прием на коммерческой основе. Доступен всем пациентам без ограничений.

  • Живая очередь или запись
  • Не требует прикрепления
  • Оплата на месте

Договорной Договорные организации

Обслуживание сотрудников компаний по договору (Узеньмунайгаз, КазМунайГаз, транспортные компании и др.).

  • Запись через модуль «Платные услуги → Договор с организацией»
  • Списки сотрудников обновляются по почте
  • У каждого сотрудника до 3 членов семьи

Соисполнение Направления из других клиник

Пациенты направляются из других поликлиник через систему ФОБС на бесплатное обследование.

  • Запись приходит через КМИС автоматически
  • Прием по ОСМС
  • Пациент не прикреплен к «Нейрону»
3
Процесс записи пациента (шаг за шагом)
1 Определить тип приема

Уточнить у пациента: ОСМС / платный / договорная организация.

↓
2 Выбрать врача

Открыть расписание нужного специалиста. Проверить свободные слоты на нужную дату.

↓
3 Проверить доступ врача к типу приема

Не все врачи принимают по ОСМС. Например, цеховой врач принимает только договорные компании. Если у врача нет доступа по ОСМС — запись невозможна.

↓
4 Выбор категории в КМИС

В зависимости от типа:

Тип приема Путь в КМИС
ОСМС Регистратор → Врач → Запись (по ОСМС)
Платный Регистратор → Платные услуги → Терапия / Специалист
Договорной Регистратор → Платные услуги → Договор с организацией
↓
5 Найти пациента по ИИН

Ввести ИИН пациента для поиска в базе. Для договорных — проверить наличие в списке организации.

↓
6 Сохранить запись

Выбрать время, подтвердить. Нажать «Сохранить».

↓
7 SMS-уведомление пациенту

После сохранения пациенту автоматически приходит уведомление через приложение Damumed (push или SMS).

Отмена записи: Если пациент звонит и хочет отменить — регистратор открывает запись и нажимает «Отменить». Запись удаляется, слот освобождается.

5
Управление договорными организациями
1 Списки сотрудников

У каждой организации ведется список сотрудников (по ИИН). Хранится в Excel / системе. Количество строк: ~4329+ записей (пример: Узеньмунайгаз).

↓
2 Обновление списков (по почте)

Организация присылает обновления по email:

  • Новые сотрудники — добавить в список
  • Переводы — перевести из одного подразделения в другое
  • Увольнения — исключить (отметить красным)
  • Возраст 80+ — исключить
↓
3 Члены семьи

Каждый сотрудник может прикрепить до 3 членов семьи. При запросе — добавить в список, присылается информация по почте.

Подразделения: У крупных организаций (Узеньмунайгаз) много подразделений, каждое со своей бухгалтерией. Сотрудников переводят между подразделениями — нужно отслеживать актуальность.

Проблема: Автоответчик и выбор языка

При звонке на 46-77-11 включается автоответчик с выбором языка:

  • Кнопка 1 — Казахский язык
  • Кнопка 2 — Русский язык

БАГ При нажатии кнопки 2 (русский) — звонок не поступает в регистратуру. Звонок проходит только при нажатии кнопки 1 (казахский).

Русскоязычные пациенты выбирают русский → звонок не доходит → пациент думает, что никто не отвечает → теряем пациентов.

Решение (предложенное): Убрать автоответчик с выбором языка. Регистраторы отвечают сразу на двух языках: «Салем алейкум / Здравствуйте». Либо настроить определение языка по коду клиента (как в Beeline).

Проблема: Нерабочие часы

После 17:00 телефон отключен. Пациенты не могут дозвониться вечером / в выходные.

Решение (предложенное): Подключить мобильный номер как дополнительную линию и оставлять его включённым 24/7. Также рассмотреть автоматизацию записи через бота / приложение вне рабочего времени.

Проблема: Самостоятельная запись по ОСМС

Пациенты по ОСМС не могут самостоятельно записаться через приложение Damumed и вынуждены звонить в регистратуру.

Решение: Необходимо выяснить, почему в приложении Damumed не отображается возможность записи по ОСМС. Возможно, не настроены слоты для онлайн-записи или нет привязки врачей к ОСМС-модулю.
7
Частые вопросы и путаница пациентов

Пациенты часто путают медицинские процедуры. Регистратор должен уточнять жалобы и направлять к правильному обследованию.

Пациент говорит Что имеет в виду Правильное направление
«Хочу проверить щитовидку» Обследование щитовидной железы УЗИ + анализы на гормоны
«Хочу шланг глотать» Проверка желудка ФГДС (фиброгастродуоденоскопия)
«Снять желудок на УЗИ» Проверка желудка ФГДС — УЗИ желудок не покажет
«Проверить кишечник» Обследование кишечника Колоноскопия — УЗИ не покажет
Путает УЗИ / рентген / КТ Не знает разницу Уточнить жалобы: «Что вас беспокоит?»
Скрипт регистратора: Всегда спрашивать: «Что именно вас беспокоит?» — вместо того чтобы сразу записывать на процедуру, которую назвал пациент. Пациент может ошибаться в терминах.
